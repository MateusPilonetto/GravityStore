Propósito da Gravity Store?

A Gravity Store tem como objetivo centralizar aplicativos de código aberto, facilitando o acesso dos usuários a essas ferramentas. O projeto não possui fins lucrativos e segue em fase de desenvolvimento. Futuramente, pretendo expandir a plataforma para incluir Custom ROMs, módulos e soluções root para usuários avançados.
