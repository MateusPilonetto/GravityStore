Sobre a Gravity Store

Uma interface mais livre para um ecossistema aberto.

A antiga ideia de reunir apps, jogos e ferramentas continua aqui — sem o acoplamento ao banco anterior. As buscas usam a API oficial de pesquisa do F-Droid; cada card abre a página oficial do aplicativo para uma instalação informada e segura.
